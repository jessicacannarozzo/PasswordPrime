Password Prime
Authors: Clare Coyle, Jess Cannarozzo, Karla Spuldaro, Laura McDougall, Michelle Le

To run:
- npm install
- node server.js
- open localhost:3000/(your_request) in your favourite browser.

For any persisting errors, please see the Wiki.

Files:
server.js : implements the Express server
quote-module : handles interaction with the Quotes API and generates passwords
quote-script.js : front-end script file, allows for password practice and testing
index.html : index page with app information
quotes.html : password trial and testing page
quotes.css : stylesheet for quotes and index
